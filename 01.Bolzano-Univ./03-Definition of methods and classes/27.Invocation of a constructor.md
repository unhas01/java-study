# 생성자 호출

생성자는 `new`연산자를 사용하여 객체가 생성될 때 런타임 지원(JVM)에 의해 자동으로 호출된다. 

```java
Person p = new Person("John", "London");
```

`Person` 클래스 객체를 생성하고 매개 변수로 전달된 값에 대해 필드 `name`, `residence`를 명시적으로 초기화하는 생성자를 호출한다. 그러면 새로 생성된 객체에 대한 참조가 변수 `p`에 할당된다.

> 참고: `new` 연산자는 생성자를 사용하여 객체를 생성하고 해당 객체에 대한 참조를 반환한다.
> 
> 모든 생성자를 클래스의 `public` 필드로 선언하는 것이 중요하다. `private`로 선언된 경우 객체를 만드려고 하면 오류가 발생한다.
